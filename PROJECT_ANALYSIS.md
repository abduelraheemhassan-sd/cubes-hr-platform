# تحليل مشروع CubesHR

## نظرة عامة
منصة متكاملة لإدارة الموارد البشرية توفر لوحة تحكم شاملة لإدارة الموظفين والعقود والوثائق والحضور والموافقات.

## الأقسام الرئيسية

### 1. لوحة الموارد البشرية (Dashboard)
- **الإحصائيات الأربعة الرئيسية:**
  - الموظفون النشطون (10)
  - تغطية الحضور اليوم (30%)
  - الموافقات المعلقة (2)
  - مخاطر الوثائق والعقود (12)
  
- **شريط المتابعة التنبيهي:**
  - موافقات معلقة (2)
  - استثناءات حضور (3)
  - وثائق ناقصة (9)
  - عقود قاربت الانتهاء (2)

- **حالة النظام:**
  - اكتمال ملفات الموظفين (100%)
  - إدخال الحضور (30%)
  - الوثائق المطلوبة (25%)

- **الرسومات البيانية:**
  - رسم بياني خطي للموظفين الجدد
  - رسم بياني دائري لتفاصيل الحضور
  - رسوم بيانية إضافية للإحصائيات

### 2. الموظفون (Employees)
- إدارة بيانات الموظفين
- عرض وتحديث المعلومات الشخصية

### 3. الهيكل التنظيمي (Organization)
- عرض الهيكل التنظيمي للشركة
- علاقات التبعية والأقسام

### 4. العقود (Contracts)
- إدارة عقود الموظفين
- تتبع تواريخ الانتهاء والتجديد

### 5. الوثائق (Documents)
- إدارة الوثائق والملفات
- تتبع الوثائق الناقصة

### 6. الموافقات (Approvals)
- عرض الموافقات المعلقة
- اتخاذ القرارات (موافقة/رفض)

### 7. الحضور والغياب (Attendance)
- تسجيل الحضور اليومي
- تتبع الاستثناءات والإجازات

### 8. الإعدادات (Settings)
- إعدادات النظام والمستخدم

## المتطلبات التقنية

### المكتبات المستخدمة:
- Tailwind CSS 4 - للتصميم
- Chart.js - للرسومات البيانية
- Cairo Font - لدعم اللغة العربية
- React 19 - للتطبيق
- shadcn/ui - للمكونات

### الألوان الأساسية:
- Brand Color: #4f46e5 (Indigo)
- Green: للحالات الإيجابية
- Orange: للتنبيهات
- Red: للأخطاء والمخاطر
- Gray: للنصوص والخلفيات

## الميزات الرئيسية

1. **البحث الذكي** - البحث عن موظف، عقد، أو طلب
2. **التنبيهات** - نظام إشعارات Toast
3. **الرسومات البيانية** - عرض بيانات ديناميكية
4. **الواجهة ثنائية الاتجاه** - دعم كامل للغة العربية
5. **التنقل السلس** - تبديل سلس بين الأقسام
