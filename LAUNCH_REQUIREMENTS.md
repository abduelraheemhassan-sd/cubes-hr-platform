## 1️⃣ المتطلبات الفنية

### أ. البنية التحتية

#### السيرفرات والاستضافة
- **خادم الويب الرئيسي**: Node.js + Express
  - الحد الأدنى: 2GB RAM, 2 CPU cores
  - الموصى به: 4GB RAM, 4 CPU cores
  - التخزين: 50GB SSD (قابل للتوسع)
  
- **قاعدة البيانات**: MySQL 8.0+
  - الحد الأدنى: 2GB RAM
  - التخزين: 100GB (قابل للتوسع)
  - النسخ الاحتياطية: يومية

- **خدمات إضافية**:
  - Redis (للـ caching): 1GB RAM
  - S3 (للتخزين): حساب AWS أو بديل
  - CDN: CloudFlare أو Akamai

#### المتطلبات البرمجية
```
Node.js: v18+
npm/pnpm: v8+
React: 19+
TypeScript: 5.9+
Drizzle ORM: 0.44+
tRPC: 11+
```
