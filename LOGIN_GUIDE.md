## 🚀 الخطوات الأولى بعد تسجيل الدخول

### 1️⃣ استكشاف لوحة المعلومات

```
من الصفحة الرئيسية، يمكنك رؤية:
- إجمالي الموظفين
- عدد الموافقات المعلقة
- نسبة الحضور
- الموظفين الجدد هذا الشهر
```

### 2️⃣ الذهاب إلى قسم الموظفين

```
1. من القائمة الجانبية، انقر على "الموظفين"
2. ستظهر قائمة بجميع الموظفين
3. يمكنك:
   - البحث عن موظف
   - فلترة حسب القسم
   - إضافة موظف جديد
   - عرض تفاصيل الموظف
```

### 3️⃣ تسجيل الحضور

```
1. من القائمة الجانبية، انقر على "الحضور"
2. اختر التاريخ (اليوم افتراضياً)
3. اختر الموظف
4. انقر على "حاضر" أو "غائب"
5. انقر على "حفظ"
```

### 4️⃣ إدارة الموافقات

```
1. من القائمة الجانبية، انقر على "الموافقات"
2. ستظهر قائمة الطلبات المعلقة
3. يمكنك:
   - عرض تفاصيل الطلب
   - الموافقة على الطلب
   - رفض الطلب
   - إضافة ملاحظات
```

### 5️⃣ عرض التقارير

```
1. من القائمة الجانبية، انقر على "التقارير"
2. اختر نوع التقرير:
   - تقرير الحضور
   - تقرير الموظفين
   - تقرير الموافقات
3. اختر الفترة الزمنية
4. انقر على "عرض" أو "تصدير"
```

## 🆘 مشاكل شائعة عند تسجيل الدخول

### المشكلة 1: "بيانات الدخول غير صحيحة"

**السبب:** البريد أو كلمة المرور خاطئة

**الحل:**
```
1. تأكد من كتابة البريد بشكل صحيح
   - admin@cubeshr.com (صحيح)
   - admin@cubeshr (خطأ - ناقص .com)
   - Admin@cubeshr.com (خطأ - حرف كبير)

2. تأكد من كتابة كلمة المرور بشكل صحيح
   - admin123 (صحيح)
   - admin (خطأ - ناقص 123)
   - Admin123 (خطأ - حرف كبير)

3. استخدم الحسابات المذكورة أعلاه فقط
```

---

### المشكلة 2: "خطأ في الاتصال بقاعدة البيانات"

**السبب:** قاعدة البيانات لا تعمل

**الحل:**
```
1. تأكد من تشغيل MySQL
   - على Windows: ابدأ MySQL من Services
   - على Mac: brew services start mysql
   - على Linux: sudo systemctl start mysql

2. تأكد من تشغيل الخادم
   - اكتب: pnpm dev

3. تحقق من الاتصال بقاعدة البيانات
   - mysql -u cubes_user -p cubes_hr_db
```

---

### المشكلة 3: "الصفحة لا تحمل"

**السبب:** الخادم لا يعمل

**الحل:**
```
1. تأكد من تشغيل الخادم
   - اكتب: pnpm dev

2. تأكد من الرابط الصحيح
   - http://localhost:3000 (محلي)
   - https://cubeshr-fqir6ovu.manus.space (Manus)

3. حاول تحديث الصفحة
   - اضغط: F5 أو Ctrl+R

4. امسح ذاكرة التخزين المؤقت
   - اضغط: Ctrl+Shift+Delete
```

---

### المشكلة 4: "رسالة خطأ غريبة"

**الحل:**
```
1. افتح أدوات المطور (F12)
2. انقر على تبويب "Console"
3. ابحث عن رسائل الخطأ
4. انسخ الخطأ وابحث عنه في الدليل
```
